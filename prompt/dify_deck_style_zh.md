# Dify 幻灯片视觉风格提示词（中文）

设计一套 **Dify** 品牌的极简幻灯片，整体风格为「杂志排版 + 工程感」。

## 品牌与配色

- 品牌主色为 **Dify 蓝 `#0033FF`**（潘通专色）。
- 主配色为 **白 + 黑 + Dify 蓝**。
- 其他颜色只出现在照片或技术示意图中，不能抢过 Dify 蓝的视觉主导地位。
- 不使用渐变和炫彩效果，整体配色干净、克制。
- Dify 蓝以整条色带、大色块、分隔条等形式出现（顶栏、底栏、章节标题条、重点高亮块）。

## 字体与文字

- 使用粗体几何无衬线字体。
- 标题：字号极大，可全大写，左对齐，行距偏紧。
- 正文：字号较小，排版整洁，短段落或简短 bullet。
- 强调「超大标题 + 小号正文」形成强烈层级对比。

## 版式系统

- 严格网格，大量留白，每页元素尽量少。
- 所有页面使用统一的页眉 / 页脚结构：
  - **右上角**：Dify Logo。
  - **左下角**：文档类型 / 作者信息 / 章节名（如 “Pitch Deck”、“Speaker: Peter Han”）。
  - **中下方居中**：版权信息（如 “Copyright © Dify Tech Inc. All Rights Reserved.”）。
  - **右下角**：年份 + 页码或日期。
- 主要内容（标题、正文、图片、示意图）位于页脚上方的中部网格区域。
- 典型构图为：**一个大标题 + 一个主视觉（照片或示意图）+ 极少文字**。

## 图片风格（通用场景）

- 使用有电影感和质感的图片。
- 支持黑白图和彩色图。
- 彩色图片需简洁、克制，可偏单色或低饱和度，让 **Dify 蓝 `#0033FF`** 始终是整个版面中最有识别度的颜色。
- 主题示例：皮肤与盲文点、布料等肌理特写，抽象物体，电影感画面，隐喻 AI、意图、人机互动和基础设施的场景等。

## 工程 / 技术培训页面的示意图风格

- 针对工程技术题材（如架构、部署、性能、Workflow、API、DevOps 等），页面可以使用**彩色技术示意图**：
  - 架构图、服务拓扑图
  - 数据流 / 时序图（sequence diagram）
  - 组件拆解图、UI 线框图
  - 代码块、调用链路图
- 示意图可以使用多种颜色区分不同组件 / 服务 / 层级 / 数据通路，但要满足：
  - 背景为白色；
  - 线条与标注清晰易读；
  - **Dify 蓝 `#0033FF` 仍是标题、关键连线、高亮标注、章节标签等的主品牌色**。
- 辅助色建议使用工程感配色（适度的绿色、橙色、灰色等），避免过于卡通的霓虹色 / 粉色。
- 技术图示页依然遵循统一的网格、页眉页脚和字体体系，保证视觉语言一致。

## 典型页面布局模式

- **封面 / 开场页**：大号 Dify Logo + 堆叠式大标题 + 小号描述 / 保密声明。
- **概念 / 价值说明页**：粗体大标题 + 简短要点 + 一张强视觉照片。
- **工程 / 架构说明页**：技术主题标题 + 中央彩色架构 / 流程图 + 简短 bullet。
- **致谢页**：左侧超大 “THANK YOU!” 文本 + 右侧联系方式与黑白二维码。

## 整体气质

- 大胆、自信、克制，偏高级「杂志风 + 工程风」。
- 更像精心设计的 editorial layout，而不是普通企业模版。
- 不使用小图标、复杂插画和渐变，只使用矩形、文字、Dify 蓝、受控的示意图配色和高质感图片。
